Data Cleaning (Delete unused columns, delete duplicates, and modify the structure)¶

At this stage i will: 1- Delete unused columns (cast, homepage, tagline, keywords, etc) 2-Delete redundancy 3-Delete null or 0 values

# After discussing the structure of the data and any problems that need to be
#   cleaned, perform those cleaning steps in the second part of this section.
#Delete unused columns (cast, homepage, tagline, keywords, etc)
df=df.drop(['keywords', 'imdb_id', 'tagline', 'cast', 'overview', 'homepage'], axis=1)
#Delete redundancy
sum(df.duplicated())
df.drop_duplicates()
#Delete null or 0 values
df.dropna(inplace=True)
#replacing 0 values with NAN
df.replace(0, np.NAN)
#handeling missing values
df['budget'] = df['budget'].replace(np.NAN,df['budget'].mean())
df['revenue'] = df['revenue'].replace(np.NAN,df['revenue'].mean())
df['runtime'] = df['runtime'].replace(np.NAN,df['runtime'].mean())
